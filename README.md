# Build instructions

Clone the repository

    $ git clone https://github.com/skrakau/sim_iCLIP.git
    $ cd sim_iCLIP

Create a build directory, configure the build and compile

    $ mkdir build
    $ cd build
    $ cmake ../src
    $ make

Requirements

 - C++14 compliant compiler
 - cmake 3.0 or newer
